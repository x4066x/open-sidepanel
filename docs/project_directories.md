# プロジェクトディレクトリ構造

```markdown
.
├── docs/                     # ドキュメント及び設計書、バージョンごとの開発手順を保存
│   └── v0.1.0/              # バージョン別の開発ドキュメント
├── entrypoints/             # WXTが要求する拡張機能タイプ別のエントリーファイル
├── public/                  # ビルド後にそのままコピーされる静的ファイル、画像等の素材
└── src/                     # ソースコード
    ├── components/          # UIコンポーネント
    │   ├── features/       # 機能単位のUIコンポーネント
    │   └── common/         # 共通UIコンポーネント
    ├── core/               # ビジネスロジック
    │   ├── messaging/      # サイドパネル、バックグラウンド、コンテンツスクリプト間通信
    │   ├── providers/      # WebAPI実装
    │   ├── services/       # バックグラウンド/サイドパネルのビジネスロジック
    │   └── storage/        # ストレージ関連の実装
    ├── types/              # インターフェース、型定義
    └── utils/              # 開発用ユーティリティ関数
```
